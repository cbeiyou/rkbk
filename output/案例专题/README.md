## 学习建议

1. **先读答题技巧**：第一篇答题技巧是万能钥匙，掌握答题方法和拿分技巧
2. **按专题学习**：每个专题对应一个知识领域，可结合章节演练的知识点巩固
3. **题目+答案对照**：先做题目，再看答案，分析答题思路和得分点
4. **重专题排序**：
   - 高频考点：整合管理、范围管理、质量管理、配置/变更管理
   - 中频考点：风险管理、采购管理、资源管理
   - 低频考点：进度管理、成本管理、沟通管理

## 万能钥匙速记

详见 `题目/第一篇答题技巧.md`，核心要点：

- **沟通不好**：任何案例都可写
- **计划没做好**：任何案例都可写
- **技术人员做PM**：角色定位不好，需要培训
- **身兼数职**：工作负荷过载
- **新技术**：风险+培训
- **变更**：书面申请+审批确认+跟踪
- **验收不通过**：验收标准未确认
- **质量问题**：缺少标准+体系+流程
